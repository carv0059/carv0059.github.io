    A: <i> is the odd one out. The other tags define an aspect of the webpage and <i> is a style element that makes the contents of the tag show up in italics.

  7. Q: What element is used to create images? Name its attributes.

    A: The <img> tag is used to create an image and its attributes are src (source) which specifies the file path to the image and alt which specifies alternate text for the image if the image can't be displayed.

  8. Q: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

  10. Q: What is the use of name and value attributes? Give two examples

    A: The name attribute is unique to the element and is used for javascript styling or to reference form data after a form is submitted. The value attribute defines the initial (default) value of the input field. For example, in a form with a drop down menu the name for each option would be the same, ie: country, language and the value for each option would be the specific countries or languages.

  12. Q: How would you link an external style sheet? Give example.

  13. Q: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    A: Within the HTML i would give the <div> element the class name "block". In the CSS i would target that class and apply the styling to it:
    .block {
    position: absolute;
    width: 200px;
    height: 200px;
    top: 50%;
    left: 50%;
    }

  14. Q: Create CSS keyframes block called "spin" that rotates an element.

    A: div { width: 100px; height: 100px; background: red; position: relative; animation: spin 5s infinite; } @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } }

  15. Q: Create CSS animation block to rotate an element that has the following properties : class "rotate-me" It rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin"

    A: .rotate-me { width: 100px; height: 100px; background: green; position: relative; animation: spin 0.5s 5; } @keyframes spin { transform: rotate(360deg); transition: linear 0.5s; }
